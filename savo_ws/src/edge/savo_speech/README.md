
## Configuration profiles

The package uses validated base parameter files and profile overlays.

Available profiles:

- `dryrun_no_hardware`
- `bench_respeaker_audio`
- `robot_savo_online_first`
- `robot_savo_full_hybrid`

The default base configuration does not access hardware, load AI models, or
contact SavoMind.
